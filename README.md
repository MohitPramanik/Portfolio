# Mohit Kumar Pramanik — Portfolio

A modern, animated personal portfolio showcasing experience, featured projects, and technical skills.

## Highlights

- Single-page layout with smooth section navigation (Home, Experience, Projects, Skills)
- Motion/interaction design using Framer Motion
- Iconography via Lucide
- Floating resume download button

## Tech Stack

- React 19 + TypeScript
- Vite
- Framer Motion
- Lucide React

Live Link: https://mohitpramanik.github.io/Portfolio/
